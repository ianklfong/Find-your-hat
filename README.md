# Find your hat
"Find your hat" is an interactive terminal game. 
>**Find your hat** is a project by "[**codecademy**](https://www.codecademy.com/learn)".

## Technologies Used
- Javascript

## Module Used
- [terminal-kit](https://github.com/cronvel/terminal-kit)
